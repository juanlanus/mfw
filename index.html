General concepts

MyFeedback (MF) is a generalized system to store and manage the evaluations and feedback information of a company.

It allows its users to set and check quantitative goals, and to evaluate or be evaluated according to the company preferences.

The system is flexible enough to allow the company to set its preferences, according to its internal culture.

It can be used as a traditional evaluation system, with quantitative or qualitative goals, and as a more modern continuous evaluation system. The latter, albeit deemed as preferable was inpractical to run because there was no suitable application. Until now.

Using a computer system to store the evaluations data is a step towards the totally paperless office. Aditionally, having the data stores in a computer enables the company to run reports on it, that can be used both to make the employees happier and more productive.

operation

The company states its preferences on how to run the evaluations of its employees; those preferences are set in the MF system configuration.

After the system is configured, it behaves the way the company wants.

The appropriate employees start doing the evaluations tasks, like setting their own or their employees objectives, informing the goals execution scores, etc.

When the due dates come (according to the company's calendar) the system calculates the employees' results and notifies them.
