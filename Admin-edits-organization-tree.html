Admin edits organization tree

 

Admin interactively defines "reports to" relationships, interacting with the current org tree and the users database.

If the edition involves orphaning any users (like for example if their manager is reassigned or leaving) they are moved to a special branch from where they have to be reassigned.

While this special branch is not empty a report is triggered daily.

Initially the special branch contains all the users, available to be assigned to their position in the hierarchy.
